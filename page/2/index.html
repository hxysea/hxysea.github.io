Hexo & Markdown 2 get your Style

抓住刹那即逝的灵感，键盘敲打，自是一道风景。

　　所需不多，逐步操作，水到渠成。

  1. 登录Git官网，选择Windows版下载安装（版本选择最新，目前最新的为2.9.0）。
  2. 登录NodeJS官网，选择V4.4.7LTS版本下载安装。

  3. 本地安装Hexo，新建目录（如MyBlog），在目录下点击鼠标右键，选在Git bash，输入命令npm install -g hexo-cli

  4. 在Git bash下输入hexo version，校验Hexo安装是否成功。

  5. 初始化hexo环境，输入hexo init .，文件目录如图。

  6. 输入命令hexo server，启动服务，输入 http://localhost:4000 可访问首页。

  7. 修改_config.yml文件， 发布到GitHub上。

  8. 输入明林hexo generate和hexo deploy将博客内容发布到Github上。

常用hexo命令

Create a new post

Run server

Generate static files

Deploy to remote sites
