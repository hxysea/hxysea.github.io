LeetCode OJ

问题描述

  • Sort a linked list in O(n log n) time using constant space complexity.

问题分析

  1. 需要做什么？（定义问题）

    • 首先看一下ListNode的定义：
    • 了解到什么(问题域)：
      • 一个链表节点包含一个int的成员变量val和一个链表引用next，通过此next引用构成一个链表。
      • 对此链表排序的排序基准即为val这个成员变量

  2. 怎么做？

    • 设计算法

      • 链表排序问题转换为其他数据结构排序实现

      • 重构链表之间的关系

      • 算法详情：

        1. 遍历链表节点将链表节点存储在数组中
        2. 根据val变量的大小将数组排序
        3. 遍历数组重构链表的先后关系，最后一个节点指向null
        4. 返回head节点

    • 选择数据结构

      • 使用数组存储链表节点
