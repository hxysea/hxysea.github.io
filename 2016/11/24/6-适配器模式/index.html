6.适配器模式

  • structure design patterns
design pattern

Adapter Desgin Pattern

  1. What is it?

    举个例子，手机充电需要3V，但是插座上提供的是220V，手机充电器就充当了适配器（转换）的角色，将220V转换成了3V供手机充电，从而将手机和插件两个不相关关的事物联系了在一起。
    简而言之，适配器模式就是通过一个适配器对象将A接口转换成B接口需要的形式。

  2. How to implemnt it?

    • 类适配， 即继承方式来包含Source类

    • 对象适配， 即使用组合方式来包含Source对象

  3. Code Demo

  4. Benefits
    给你想要的，适配一次即可

    • java.io.InputStreamReader(InputStream){return Reader}
    • java.io.OutputStreamWriter(OutputStream){return Writer}
