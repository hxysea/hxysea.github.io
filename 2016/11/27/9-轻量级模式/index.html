9.轻量级模式

  • structure design patterns
design pattern

Flyweight Pattern

  1. What is it?

    如果需求环境内存受限（比如手机和嵌入式设备），而且创建对象比较耗内存且创建时间较长，则可提前创建好以供复用。线程池、数据库连接池以及字符串池都是其思想的应用，简而言之，将所需对象提前创建好以供共享使用即为轻量级模式。

  2. How to implement it?

    • 分离目标对象属性，分为外部标识属性和对象本征属性。
    • 设计目标对象接口及实现
    • 设计工厂类，提供根据对象外部标识属性获取目标对象方法

  3. Code Demo

  4. Important Points

    • 根据需求，强制客户端使用轻量级工厂创建对象
    • 根据需求对创建对象的时间和空间进行折中考虑
    • 不适用于目标对象本征属性过多的情况，因为会增加创建复杂性
  5. JDK Example
    • wrapper Class#valueOf()，如Integer的valueOf方法
